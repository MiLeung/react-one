# react-one

> A single store, single action, global state management system for React apps.

[![NPM](https://img.shields.io/npm/v/react-one.svg)](https://www.npmjs.com/package/react-one) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save react-one
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'react-one'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
